**Project Overview:**

This repository houses a comprehensive Customer Relationship Management (CRM) system developed using Spring Boot. The main focus of this project is to provide robust functionalities for managing customer interactions and data. 

**Key Features:**
- **REST APIs:** Developed over 20 REST APIs to facilitate seamless communication between the client-side and server-side components.
- **Database Configuration:** Implemented database configuration using Hibernate to ensure efficient data storage and retrieval.
- **CRUD Operations:** Enables Create, Read, Update, and Delete operations for managing customer data effectively.
- **Authentication and Authorization:** Implemented secure authentication and authorization mechanisms to safeguard sensitive information.
- **Scalability:** Designed with scalability in mind to accommodate future expansion and increased user load.
